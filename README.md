letvcloudchecker
================
Yet another batch checker of video status on Letvcloud.

Able to save the result in csv file.

Usage
------
python3 letvcheck.py (-h)( -s) vu1 vu2 ...

-s: Save to CSV file. -s save.csv

-h: Help.

Requirement
-------
- Python 3.3
